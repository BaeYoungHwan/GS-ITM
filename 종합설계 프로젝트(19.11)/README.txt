2019 정보통신공학과 1조 종합설계

< 주제 >
웹데이터 및 작물데이터를 활용한 지능정보시스템

<팀명>
 아이들

<팀원>
 지도교수 : 여 현
 하드웨어: 김승재, 김현준
 분석:  배영환(조장), 강주완
 소프트웨어: 김용군, 정다영

< 구성 내용 >
시중에 판매/재배되는 작물들의 종류는 굉장히 다양하며, 다수 작물들의 생장 조건을 파악하기 위해서는 규격화된 작물의 데이터가 필요하다. 이에 작물을 직접 재배하고 적합한 환경데이터를 분석하여 가치있는 데이터를 만든다.

< Tool >
Eclipse ( Web )
Jupyter ( Python )
R Studio ( R )
Arduino ( Arduino )

< 한 일 >
조장으로서 전체적인 구상 및 PPT, 데이터 분석, Web 디자인, 발표를 담당.

1. 전체적인 구상
구상의 핵심은 빅데이터 분석으로서 Hadoop을 이용한 대용량 데이터 분석을 계획 했지만 여건이 되지 않아 저희들이 직접 재배하여 Arduino를 통해 센서 수집후 GitHub로 파일관리, SPSS 데이터 분석을 실시하였다.

2. 데이터 분석
데이터 분석은 Web에서 크롤링해온 데이터와 직접 재배한 환경데이터를 가지고 분석을 실시하였고 SPSS와 Python 을 가지고 분석을 진행했다.

3. Web 디자인
전체적인 틀은 템플릿을 이용하여 제작되었으며 메뉴, 데이터, 사진 등을 변경하여 Web Page를 제작 했다.

< 문제점 >
데이터 분석을 하면서 통계적으로 부족한 점을 느껴 4학년 2학기임에도 불구하고 수교과에 찾아가 통계학 수업을 들으며 기초를 다졌다. 또한 여러 분석 기법을 알아보기 위해 공공데이터 분석 수업을 들으며 여러가지 데이터 분석 방법을 학습하였다.

< 배운점 > 
1학기에는 Cloudera를 이용한 Hadoop 시스템을 공부하였고, 논문 읽는법, 찾는법 등 연구실에서 대학원 선배님에게 많은 가르침을 받았다. 2학기에는 직접 재배를 하며 센서를 통해 데이터 수집도 해보고 R, Python을 이용하여 데이터 분석 및 워드클라우드, 크롤링 등 여러 작업을 진행하였다.
